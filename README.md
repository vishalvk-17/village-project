# Village Project

Village Project is a full-stack web application built to manage villagers,
and user profiles for a village-level digital portal.

## Features
- Villager registration and profile management
- Login and logout with session handling
- Image upload using Multer
- Role-based access (Admin / Viewer)

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS
- Multer

## Project Structure
- models/    → Database schemas
- routes/    → Application routes
- views/     → EJS templates
- utils/     → Utility files (multer, helpers)
- public/    → Static files (CSS, assets)
- app.js     → Main application file
